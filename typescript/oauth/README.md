# OAuth 2.0

This example demonstrates how to integrate the TIgol OAuth 2.0 system with a minimal Express.js web server.

## Running the Server

### Using npm
```sh
npm install
npm run server.ts
```

### Using yarn
```sh
yarn install
yarn run server.ts
```

### Using pnpm
```sh
pnpm install
pnpm run server.ts
```

### Using bun
```sh
bun install
bun run server.ts
```

The server entry point is `server.ts`.
